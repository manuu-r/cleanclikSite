1. Introduction

This Privacy Policy explains how MaybeCoded ("we," "us," "our," or "Company") collects, uses, stores, processes, and protects your personal information when you use the Sparcd mobile application ("App").

We are committed to protecting your privacy and ensuring transparency about our data practices. By using the Sparcd App, you acknowledge that you have read and understood this Privacy Policy and consent to the collection and use of your information as described herein.

2. Information We Collect

2.1 Information You Provide Directly

  • Email address

  • Authentication method (email/password, Google Sign-In, or Apple Sign-In)

Demographic Information (Optional but Recommended):

  • Date of birth
  • Gender
  • Occupation
  • Country/Location

2.2 Psychological and Behavioral Profile Data

Psychological Profile:

  • Tendency Classification (Four Tendencies framework by Gretchen Rubin)
  • Journey Stage (Dreamer, Planner, Doer, or Lifestyle Builder)
  • Mindset Indicators (growth vs. fixed mindset)
  • Processing Style (rumination score, stimulation tolerance)
  • Attachment Motivation and feedback preferences
  • Failure Response patterns

Behavioral Signals:

  • Optimal engagement times and notification response rates
  • Current and historical streaks
  • Energy phase tracking and stress cycle indicators
  • Feature usage frequency and preferences

2.3 Goal, Task, and Personal Content Data

  • Goal descriptions, objectives, and success criteria
  • Tasks, micro-actions, and milestone definitions
  • Task completion status and timestamps
  • Personal reflections, journal entries, and notes
  • Progress tracking data and updates

2.4 AI Memory and Context Data

The App uses Mem0-powered AI memory system to store:

  • Conversation history and coaching interactions
  • User preferences and patterns identified over time
  • Past achievements and goal history context
  • Personal preferences regarding communication style
  • Contextual information about life circumstances and challenges

2.5 Automatically Collected Technical Information

Device Information:

  • Device type (iPhone, iPad, Android)
  • Operating system and version
  • Device identifiers (IDFA, Android Advertising ID)
  • Device model and manufacturer

Connection and Usage Data:

  • Internet Protocol (IP) address
  • Mobile network operator information
  • Pages visited and features accessed
  • Time spent on features and sessions
  • Error messages and crash reports
  • Performance metrics

Location Information:

  • General location data derived from IP address
  • Precise GPS location data (only if you grant explicit permission)

4. Data Storage and Security

4.1 Where We Store Your Data

  • Primary Database: MongoDB Atlas (cloud database)
  • AI Memory: MongoDB Atlas with embedding vectors
  • Backups: Encrypted backups in secure data centers
  • Processing: Google Cloud services for AI and embeddings

4.2 Security Measures

  • Encryption for data in transit (TLS/SSL) and at rest
  • Industry-standard password hashing with salting
  • Role-based access control
  • Multi-factor authentication for administrator accounts
  • Comprehensive logging of data access
  • Periodic security testing and vulnerability assessments
